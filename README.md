# K_DOCS — Documentation Module

Implementation module for the K_DOCS web documentation system.

## Structure

- `Knowledge/K_DOCS/` — Module implementation (scripts, configs, conventions)
- `docs/` — Web-facing output directory (GitHub Pages compatible)

## Relationship

- `Knowledge/K_MIND/` — Memory system (sessions, mindmap, memory scripts)
- `Knowledge/K_DOCS/` — Documentation system (viewer, publication tools)
- `docs/` — Published content served by GitHub Pages
