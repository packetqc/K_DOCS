## Bienvenue

**Knowledge 2.0** est la deuxième génération du système d'Intelligence d'Ingénierie IA. Il introduit un protocole d'initialisation de session structuré qui remplace les prompts non structurés par un questionnaire déterministe, un routage de commandes, des skills composables et une déduplication de méthodologie.

#### Publications

Publications techniques documentant l'architecture du système, les décisions de conception et les détails d'implémentation.

[Parcourir les publications &rarr;]({{ site.baseurl }}/fr/publications/)

#### Code source

Dépôt open source avec fichiers de méthodologie, scripts, skills et le cadre complet de knowledge.

[Voir sur GitHub &rarr;](https://github.com/packetqc/knowledge)

---

## Fonctionnalités clés

| Fonctionnalité | Description |
|----------------|-------------|
| **Questionnaire de session** | Grille d'intégration structurée (A1-D3) valide le contexte avant le travail |
| **Routeur de commandes** | `routes.json` mappe l'intention aux programmes — exécution déterministe |
| **Architecture de skills** | `SkillRegistry` composable avec skills typés et testables |
| **Résolution de méthodologie** | Chargement automatique par famille — déclarer une famille, obtenir tous les fichiers |
| **Moteur de déduplication** | Suivi au niveau session — réduction de 66% des lectures répétées |
| **Modes interactifs** | Conception, Documentation, Diagnostic — chacun avec son patron de phases |
| **Persistance de checkpoint** | Checkpoints pré-exécution survivent aux crashes et à la compaction |
