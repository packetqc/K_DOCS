## Welcome

**Knowledge 2.0** is the second generation of the AI Engineering Intelligence system. It introduces a structured session initialization protocol that replaces unstructured prompts with a deterministic questionnaire, command routing, composable skills, and methodology deduplication.

#### Publications

Technical publications documenting the system architecture, design decisions, and implementation details.

[Browse publications &rarr;]({{ site.baseurl }}/publications/)

#### Source Code

Open source repository with methodology files, scripts, skills, and the complete knowledge framework.

[View on GitHub &rarr;](https://github.com/packetqc/knowledge)

---

## Key Features

| Feature | Description |
|---------|-------------|
| **Session Questionnaire** | Structured onboarding grid (A1-D3) validates context before work begins |
| **Command Router** | `routes.json` maps intent to programs — deterministic execution |
| **Skill Architecture** | Composable `SkillRegistry` with typed, testable skills |
| **Methodology Resolution** | Family-based auto-loading — declare a family, get all matching files |
| **Deduplication Engine** | Session-level tracking — 66% reduction in repeated file reads |
| **Interactive Modes** | Conception, Documentation, Diagnostic — each with its own phase pattern |
| **Checkpoint Persistence** | Pre-execution checkpoints survive crashes and context compaction |
