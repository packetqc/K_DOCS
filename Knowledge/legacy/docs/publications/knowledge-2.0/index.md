## Overview

Knowledge 2.0 introduces an **Interactive Intelligence Framework** — a structured questionnaire that runs at the start of every session, validates context, routes commands programmatically, loads only the relevant methodology, and opens the right working mode.

### Key Features

| Feature | Description |
|---------|-------------|
| **Session Questionnaire** | Structured onboarding protocol (A1-D3 grid) |
| **Command Router** | `routes.json` maps intent to programs |
| **Skill Architecture** | Composable `SkillRegistry` with typed skills |
| **Methodology Resolution** | Family-based auto-loading from `methodology/` |
| **Deduplication Engine** | `filter_unread` / `mark_read` — zero re-reads |
| **Interactive Modes** | Conception, documentation, diagnostic |
