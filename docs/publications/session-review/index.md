## Abstract

The Session Review interface (I1) is an interactive web page for reviewing knowledge system work sessions. Select a session from the dropdown and explore its full report across 5 sections: summary, metrics, time compilation, deliveries, and lessons learned.

Companion tool to [#20 — Session Metrics & Time Compilation]({{ '/publications/session-metrics-time/' | relative_url }}).

**[Launch Session Review (Interface I1) →]({{ '/interfaces/session-review/' | relative_url }})**

## How to Use

1. **Select a session** from the dropdown — sessions are listed by date and title
2. **Refine** by section: All, Summary, Metrics, Time, Deliveries, Lessons
3. **Browse** the session report — each section shows structured data from the session
4. **Data source**: sessions are loaded from `docs/data/sessions.json`, generated by `scripts/generate_sessions.py`

> **Version note**: Only sessions from **v52+** (2026-02-27 onwards) are listed. Earlier sessions lack the structured protocol data (three-channel persistence, pre-save summary, real-time issue comments) required by this viewer.
