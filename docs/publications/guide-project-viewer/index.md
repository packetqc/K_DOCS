## Getting Started

The Project Viewer is a portfolio dashboard showing all registered Knowledge projects — their status, GitHub board integration, issue tracking, and progress metrics.

**To open it:**
- From the **Main Navigator**: click *I4 Project Viewer* in the left panel
- Direct URL: `/interfaces/project-viewer/`

## Project List

The main view lists all projects with:
- **P# identifier** — unique project number
- **Title** — project name
- **Status** — active, completed, paused, or archived
- **Board link** — direct link to the GitHub Project board

## Project Detail

Click a project to see:
- **Description** — project goals and scope
- **Issues** — linked GitHub issues with status
- **Progress** — completion percentage based on issue states
- **Timeline** — project milestones and dates

## Tips

- **Filter by status**: use the dropdown to show only active or completed projects
- **Print**: the interface has print-optimized styles for portfolio reports
- **Data source**: project data comes from `docs/data/projects.json`
