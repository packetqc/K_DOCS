## Getting Started

The Main Navigator is the central hub of the Knowledge platform. It gives you access to all interfaces, publications, methodologies, configurations, and project data — from a single page.

**To open it:** navigate to `/interfaces/main-navigator/` or click *I2 Main Navigator* from any interface listing.

## Three-Panel Layout

| Panel | Position | Purpose |
|-------|----------|---------|
| **Left** | Left sidebar (220px) | Widget directory — browse all content sections |
| **Center** | Middle (fills space) | Interfaces — interactive tools load here |
| **Right** | Right sidebar (hidden by default) | Document viewer — publications, guides, JSON configs |

**Resizing:** drag the dividers between panels, or click a divider to cycle through preset sizes.

## Left Panel Widgets

The left panel contains collapsible widget sections:

- **Interfaces** — launch I1–I5 in the center panel
- **Documentation** — user guides, admin guides, quick starts
- **Essentials** — key reference documents
- **Commands** — grouped command reference
- **Methodologies** — operational methodology files
- **And more** — hubs, profile, publications, stories, configurations

Click any item to load it. Most items open in the right panel (document viewer). Interfaces open in the center panel.

## Document Viewer

The right panel renders documents inline:
- **Markdown** files render with headings, tables, and diagrams
- **JSON** files render as formatted tables with clickable links
- **Tabs** track your navigation — click tabs to switch between opened documents, close with ×

## Tips & Shortcuts

- **Expand right panel**: click the right divider to cycle sizes (hidden → 50% → full → hidden)
- **Collapse left panel**: click the left divider to hide/show the widget directory
- **ℹ button**: click the info icon next to an interface to open its user guide
- **State persistence**: panel sizes, widget open/closed states, and tabs survive page reloads
