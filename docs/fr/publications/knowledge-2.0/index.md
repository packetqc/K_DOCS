## Aperçu

Knowledge 2.0 introduit un **Cadre d'Intelligence Interactive** — un questionnaire structuré qui s'exécute au début de chaque session, valide le contexte, route les commandes de façon programmatique, charge uniquement la méthodologie pertinente et ouvre le bon mode de travail.

### Fonctionnalités clés

| Fonctionnalité | Description |
|----------------|-------------|
| **Questionnaire de session** | Protocole d'intégration structuré (grille A1-D3) |
| **Routeur de commandes** | `routes.json` mappe l'intention aux programmes |
| **Architecture de skills** | `SkillRegistry` composable avec skills typés |
| **Résolution de méthodologie** | Chargement automatique par famille depuis `methodology/` |
| **Moteur de déduplication** | `filter_unread` / `mark_read` — zéro re-lecture |
| **Modes interactifs** | Conception, documentation, diagnostic |
