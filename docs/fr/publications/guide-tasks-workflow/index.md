## Démarrage

L'interface Flux de travail suit chaque tâche à travers son cycle de vie — de la réception initiale à la complétion. Chaque tâche progresse à travers des étapes définies avec des portes de validation.

**Pour l'ouvrir :**
- Depuis le **Navigateur principal** : cliquez sur *I3 Flux de travail*
- URL directe : `/fr/interfaces/task-workflow/`
- Se charge comme interface par défaut dans le panneau central

## Sélecteur de tâche

1. **Tâche** — sélectionnez une tâche spécifique ou *Toutes les tâches*
2. **Vue** — basculez entre Tableau de bord, Détail ou Validation

## Étapes du flux

| Étape | Description |
|-------|------------|
| **INITIAL** | Tâche reçue, pas encore classifiée |
| **CLASSIFIÉ** | Type d'ingénierie déterminé |
| **PLANIFIÉ** | Plan d'implémentation approuvé |
| **EN_COURS** | Développement actif |
| **REVUE** | Revue de code et validation |
| **VALIDÉ** | Toutes les vérifications passées |
| **LIVRÉ** | Commité et poussé |
| **TERMINÉ** | Tâche fermée, métriques enregistrées |

## Vues

- **Tableau de bord** — vue d'ensemble avec distribution et barres de progression
- **Détail** — informations complètes : métadonnées, historique, issues liées
- **Validation** — résultats de validation, tests et vérifications de conformité
