## Démarrage

Le Navigateur principal est le hub central de la plateforme Knowledge. Il donne accès à toutes les interfaces, publications, méthodologies, configurations et données de projets — depuis une seule page.

**Pour l'ouvrir :** naviguez vers `/fr/interfaces/main-navigator/` ou cliquez sur *I2 Navigateur principal*.

## Disposition trois panneaux

| Panneau | Position | Fonction |
|---------|----------|----------|
| **Gauche** | Barre latérale (220px) | Répertoire de widgets — parcourir toutes les sections |
| **Centre** | Milieu (remplit l'espace) | Interfaces — les outils interactifs se chargent ici |
| **Droite** | Barre latérale (caché par défaut) | Visualiseur — publications, guides, configs JSON |

**Redimensionner :** glissez les diviseurs ou cliquez pour alterner entre les tailles prédéfinies.

## Widgets du panneau gauche

Le panneau gauche contient des sections repliables :

- **Interfaces** — lancer I1–I5 dans le panneau central
- **Documentation** — guides utilisateur, guides admin, démarrages rapides
- **Essentiels** — documents de référence clés
- **Commandes** — référence groupée des commandes
- **Méthodologies** — fichiers de méthodologie opérationnelle
- **Et plus** — hubs, profil, publications, histoires, configurations

Cliquez sur un élément pour le charger. La plupart s'ouvrent dans le panneau droit.

## Visualiseur de documents

Le panneau droit affiche les documents en ligne :
- Les fichiers **Markdown** s'affichent avec titres, tableaux et diagrammes
- Les fichiers **JSON** s'affichent en tableaux formatés avec liens cliquables
- Les **onglets** suivent votre navigation — cliquez pour basculer, fermez avec ×

## Astuces

- **Agrandir le panneau droit** : cliquez le diviseur droit pour alterner les tailles
- **Replier le panneau gauche** : cliquez le diviseur gauche
- **Bouton ℹ** : cliquez l'icône info à côté d'une interface pour ouvrir son guide
- **Persistance** : tailles de panneaux, états des widgets et onglets survivent au rechargement
